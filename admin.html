Security Overview

Admin Route Protected
இந்த page load ஆகும்போதே users/{uid}.role == "admin" check பண்ணுது. இல்லன்னா access தடுக்கப்படும்.
Firestore Security Rules
firestore.rules file-ல் content collections-க்கு admin-only write, users-க்கு own-data-only access define பண்ணப்பட்டிருக்கு. Firebase Console-ல் paste பண்ணுங்க.
Storage Security Rules
storage.rules file public read, admin-only write define பண்ணுது. Firebase Console → Storage → Rules-ல் paste பண்ணுங்க.
Role Verification
Client-side check UI-ஐ மட்டும் மறைக்கும் — real protection Firestore rules-லேயே தான். Rules இல்லாம deploy பண்ணாதீங்க.
User account deletion limitation
"Delete User" இந்த panel-ல் Firestore profile-ஐ மட்டும் delete பண்ணும். Firebase Auth account-ஐ முழுசா அழிக்க Admin SDK / Cloud Function தேவை.
